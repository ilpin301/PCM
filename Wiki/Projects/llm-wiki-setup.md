# LLM Wiki Setup

The project to bootstrap a core LLM Wiki vault from an empty Obsidian folder, following the `llm-wiki-core-setup.md` guide.

## Goal

Build a clean, beginner-friendly LLM Wiki with:
- Raw source layer
- Compiled Wiki layer
- Schema and agent rules
- Note templates
- Deterministic `wiki_tool.py` tooling
- Source manifest and catalog
- Lint and audit gates

## Status

Core setup complete. Vault is functional. Ready for real sources to be ingested.
